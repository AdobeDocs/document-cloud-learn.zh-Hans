# 60秒Acrobat概述

![60秒Acrobat图像](../assets/Hero-60sec.png)

60秒的Acrobat可为您提供小型教程，帮助您在一分钟或更短的时间内学习Acrobat DC中的新技巧。 这些基于任务的提示可帮助您解锁Acrobat的一些隐藏功能，从而掌握处理PDF文件的新技能。 您可以观看其中一个视频以快速获得答案，或观看五个视频以提高文档的工作效率 — 并且仍有时间享受咖啡休息。
