# 60秒Acrobat概述

60秒的Acrobat为您提供小型教程，可帮助您在1分钟或更短的时间内学会Acrobat中的新技巧。 这些基于任务的提示可帮助您通过解锁一些Acrobat的隐藏宝石，掌握处理PDF文件的新技能。 您可以观看其中一项快速获得答案，或观看五项以提高文档工作效率，同时还有时间享受茶歇。
