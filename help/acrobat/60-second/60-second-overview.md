# 60秒Acrobat概述

![60秒Acrobat图](../assets/Hero-60sec.png)

60秒Acrobat可为您提供小小的教程，帮助您在Acrobat中在一分钟内学会新技巧。 这些基于PDF的提示可帮助您解锁某些Acrobat的隐藏项，从而获取处理任务文件的新技能。 您可以观看1以快速获得答案，或观看5以提高文档工作效率 — 同时仍有时间享受茶歇。
