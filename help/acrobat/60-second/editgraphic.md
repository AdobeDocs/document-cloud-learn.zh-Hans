# 在PDF中编辑图形

在本教程（60秒）中，了解如何使用Illustrator对PDF中的图形进行高级编辑！ 选择“编辑PDF”后，右键单击图形，然后选择在Illustrator中对其进行编辑。 完成后，只需选择“保存在Illustrator中” ，Acrobat即会更新PDF中的图稿。
