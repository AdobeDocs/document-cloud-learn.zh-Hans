# 高级任务概述

使用Acrobat中的这些高级工具，超越基础知识。 了解如何创建动作以重复平凡的任务、删除敏感信息、减少和优化大型文件、快速收集表单数据以及创建可普遍访问的PDF文件。 查看这组精心编制的[播放列表](https://experienceleague.adobe.com/zh-hans/playlists/acrobat-peform-advanced-tasks)，了解如何在Acrobat中执行高级任务。
