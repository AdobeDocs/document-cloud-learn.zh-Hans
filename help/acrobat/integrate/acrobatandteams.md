# [!DNL Microsoft Teams]中的PDF协作

Adobe Acrobat团队版允许用户通过查看、批注和审阅PDF与同事协作和工作，而无需离开[!DNL Microsoft Teams]。 此集成包括：

* 高质量查看 — 这仅是PDF查看体验，可让您直接在[!DNL Microsoft Teams]内打开高保真PDF，以获得令人满意的无缝工作体验
* 简化PDF审阅 — 在[!DNL Microsoft Teams]中直接将渠道成员的反馈收集到单个PDF中，而不是通过多个附件收集和协调反馈
* 获得进度通知 — 当其他人对您的文档采取操作时，在团队中获得通知
* 让审阅人知晓 — 当其他人回复其注释时，审阅人会收到通知
* 安全协作 — 与受信任的成员安全协作，并保留在[!DNL Microsoft SharePoint]或[!DNL OneDrive]中存储文档版本的能力
