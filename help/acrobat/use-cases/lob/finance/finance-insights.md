# 准备从洞察到交付的面向执行官的财务文档

了解财务团队如何使用Acrobat Studio通过整合财务文档、与利益相关方安全协作以及创建管理层就绪的报告和演示来从分析过渡到交付。 这一端到端的工作流程展示了如何呈现洞察力、管理审核、应用品牌一致性设计以及交付安全、可随时进行审核的财务材料，用于领导层和董事会报告。 此用例旨在按顺序完成。 每个视频均以工作流程中的上一个步骤为基础。

## 理解财务绩效

首先整合财务文档和呈现关键见解，以准备供管理层审核的信息。
