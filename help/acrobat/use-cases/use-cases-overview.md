# 用例概述

了解如何使用Acrobat提高工作效率，并将信息转变为适用于您的团队和行业的见解。

## 业务线

了解不同业务领域的团队如何使用Acrobat解决日常文档难题、简化工作流程和支持关键业务工作。

## 新增功能

>[!BEGINTABS]

>[!TAB 无瓶颈的IT启用]

了解[IT团队](lob/it/it-enablement.md)如何使用Acrobat Studio在整个组织中简化文档工作流程、增强安全性和合规性并扩展治理计划

>[!TAB 使用Acrobat Studio加速产品发布]

了解营销团队如何使用Acrobat Studio在单个PDF空间中集中化[产品发布](lob/marketing/marketing-product-launch.md)资源、简化利益相关者审阅并扩展组织就绪性。

>[!ENDTABS]
