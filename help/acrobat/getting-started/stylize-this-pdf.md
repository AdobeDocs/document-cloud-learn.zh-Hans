# 设置此PDF的风格

了解如何使用风格化此PDF通过使用由Adobe Express[Adobe Express](https://express.adobe.com)提供支持的应用程序内工具来应用PDF推荐、颜色主题、图像和设计元素，以改善字体的视觉外观。 本教程还介绍了如何优化版面、添加Adobe Stock或AI生成的图像以及自定义文本样式以创建精美专业外观的文档。 本视频教程使用[新的Acrobat体验](new-workspace.md)。

## 如何让此PDF变得风格化对各个团队有所帮助

* 法律与合规

  法律团队使用“风格化此PDF”将一致的格式、排版和视觉效果应用于政策和指导文档，使它们看起来专业且易于阅读，同时保持合规性和原创性。

* 人力资源

  HR团队使用风格化此PDF将政策文档、入门资料和培训资源转换为视觉上清晰、易于理解的PDF，以提高理解和员工参与度。

* 销售团队

  销售团队使用“风格化此PDF”，可通过应用品牌字体、颜色和图像来快速润色建议书和演示文稿，从而帮助交付客户就绪的文档，而无需返回设计工具。

* 营销与通信

  营销团队使用“风格化此PDF”，通过优化布局、应用视觉主题和引入图像来使PDF与品牌标准保持一致，从而确保素材在各营销活动中的视觉一致性。
