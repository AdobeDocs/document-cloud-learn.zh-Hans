# 将PDF导出为Microsoft Office格式

了解如何将PDF文件导出为可编辑格式，如Microsoft Word、PowerPoint和Excel。 导出为另一种文件格式后，您可以对文档进行重要的修订和更新。

>[!TIP]
>
>要将多个PDF文件导出为其它格式，请尝试使用[Action Wizard](../advanced-tasks/action.md)。
