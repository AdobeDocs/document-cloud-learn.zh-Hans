# 配置和管理“修改协议”

了解如何在Acrobat Sign中配置和管理&#x200B;**修改协议**。 在本教程中，管理员可以设置帐户级别和组级别控件，了解资格规则，并了解在发送后进行更改时如何保留协议完整性。 了解如何跟踪和审核发件人操作以支持合规性和透明度。

您将学到的内容：

* 哪些管理员设置可以启用或限制协议修改
* 协议可以和无法修改时
* 修订模式如何保护协议完整性
* 哪些更改会记录到审核报告中
