# 设置外部归档

了解如何为已签名协议的备份副本创建外部归档。 可将副本归档到任何电子邮件地址，或归档到 [!DNL Box] 和Evernote。 已签名并完成的电子邮件的副本将发送到指定的电子邮件地址或云服务。
