# 使用复选框组指导所需的选择

了解如何使用复选框组来控制签名者可以在表单上选择多少个选项。 复选框组有助于确保签名者在提交协议之前通过强制执行简单规则（例如，要求选择的最小或最大数量）正确完成文档。

了解如何设置复选框组：

* 防止提交不完整或无效
* 通过清晰的内联反馈为签名者提供指导
* 提高以表格形式收集信息的准确性
