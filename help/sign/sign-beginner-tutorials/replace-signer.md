# 替换签名者

了解如何替换签名者 — 例如，在发送文档以请求签名时使用错误的电子邮件。

>[!NOTE]
>
>替换收件人的默认设置是用新收件人完全替换当前收件人。 商业和企业级别计划还有其他可用选项。
