# 更换签名者

了解如何更换签名者，例如，在发出文档进行签名时，使用了错误的电子邮件。

>[!NOTE]
>
>替换收件人的默认设置是将当前收件人完全替换为新收件人。 商业和企业级别计划还有其他可用选项。
