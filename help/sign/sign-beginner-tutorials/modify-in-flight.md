# 发送后修改文档

了解如何修改已发送以供签名的文档 — 例如，错误发送错误的文档时。 在需要时，请使用修改协议：

* 替换或更新文档
* 更正表单字段或预填值
* 删除未完成其操作的收件人

>[!NOTE]
>
>发送后修改文档仅适用于企业级计划。 您的Acrobat Sign管理员需要在&#x200B;**帐户设置**&#x200B;的&#x200B;**发送设置**&#x200B;部分中启用&#x200B;**修改协议**&#x200B;才能使用此功能。
