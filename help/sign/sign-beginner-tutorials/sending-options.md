# 配置发送选项

了解如何在发送文档以供签名时自定义协议详细信息、附加文件以及配置其他设置，例如提醒、到期和收件人语言。

>[!NOTE]
>
>发送选项可由管理员配置，因此您可能看不到本教程中提到的完全相同的选项。
