# 何时使用创建副本

已发送协议 — 但协议已过期、已取消，或者您需要重新发送协议？ 创建副本允许您复制任何现有协议、调整需要更改的内容，并在几秒钟内重新发送协议。 无需从头开始重建。 创建副本在以下情况下可用：

* 您是协议的原始发件人
* 协议是从新版请求签名体验发送的
* 协议不包括见证收件人角色
* 由于文档保留或GDPR删除，未删除协议
* 协议不需要访问发件人不再有权访问的库模板

>[!NOTE]
>
>创建副本可用于企业级计划。
