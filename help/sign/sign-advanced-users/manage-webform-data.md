# 管理Web表单数据

了解如何跟踪、管理和导出[Web表单](webform.md)中的数据。 本教程介绍了其他参与者Web表单工作流，以及如何将多个数据条目合并为单个.csv文件以供查看和分析。

>[!NOTE]
>
>只有商业和企业级计划才能选择配置Web表单访问和签名者选项。 在帐户或组级别，可以在帐户>帐户设置>全局设置> Web表单下启用或禁用Web表单访问和签名者选项。
