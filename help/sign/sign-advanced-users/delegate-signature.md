# 使用委派人角色

了解如何使用委派人角色将文档发送给中介，然后中介可将文档路由以供签名。

>[!NOTE]
>
>委派人角色仅适用于企业级计划。 您的Acrobat Sign管理员可能需要在&#x200B;**帐户设置**&#x200B;的&#x200B;**发送设置**&#x200B;部分中启用&#x200B;**允许发送人将某些接收者标记为委派人**&#x200B;设置，才能使用此功能。
