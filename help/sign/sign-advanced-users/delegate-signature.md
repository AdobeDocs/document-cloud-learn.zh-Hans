# 使用委派人角色

了解如何使用委派人角色将文档发送给中介人，然后中介人可以路由文档以供签名。

>[!NOTE]
>
>委派人角色仅适用于企业级计划。 您的Acrobat Sign管理员可能需要启用 **允许发件人将某些收件人标记为委派人** 设置 **发送设置** 部分 **帐户设置** 才能使用此功能。
