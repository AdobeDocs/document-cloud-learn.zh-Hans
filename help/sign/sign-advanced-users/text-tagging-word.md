# 使用文本标记 [!DNL Microsoft Word]

了解如何通过将Acrobat Sign文本标记添加到 [!DNL Microsoft Word]的 当发送文档进行签名时，文本标记会自动在您的文档上创建表单字段，而无需手动放置字段。 此过程显著减少了对 [!DNL Microsoft Word] 发送以供签名的文档。
