# 使用文本标记 [!DNL Microsoft Word]

了解如何通过在以下位置添加Acrobat Sign文本标记来创建可重用的文档模板 [!DNL Microsoft Word]. 当文档发送以供签名时，文本标记会自动在文档上创建表单字段，无需手动放置字段。 此流程可显着减少修订和更改所需的时间。 [!DNL Microsoft Word] 您发送以供签名的文档。
