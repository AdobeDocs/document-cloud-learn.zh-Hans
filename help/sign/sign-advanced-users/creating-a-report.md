# 报告和交易使用情况

了解如何生成报告和跟踪交易使用情况。 这些报告可能包含已签名协议的百分比或平均签名时间等信息。 通过生成报告和跟踪使用情况，您可以了解文档签名流程。

>[!NOTE]
>
>报告仅适用于商业和企业级别计划。
