# 集成概述

![Sign集成图像](../assets/Hero-Integrate.png)

您可以在您的组织已经使用的其他应用程序(如Microsoft、Salesforce、Workday和Marketo)中使用Adobe Sign，以列举几个。 通过这些集成指南和教程，了解如何简化电子签名工作流程。

>[!NOTE]
> 如果您在访问这些功能时遇到问题，请与您组织的管理员联系，以确保集成已启用。
