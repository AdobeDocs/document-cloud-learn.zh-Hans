# 在Adobe Sign for Salesforce中创建自定义按钮

了解如何在 Salesforce.com 中创建标准对象中的自定义按钮。在此示例中，自定义按钮被添加到Salesforce中的&#x200B;**Contact**。 单击该按钮后，它将启动发送过程，并直接在协议中填充&#x200B;**Contact**&#x200B;中的信息。
