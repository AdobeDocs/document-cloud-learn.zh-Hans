# 在适用于Salesforce的Acrobat Sign中创建自定义按钮

了解如何在 Salesforce.com 中创建标准对象中的自定义按钮。在此示例中，将自定义按钮添加到 **联系人** 在Salesforce中。 单击按钮后，将启动发送过程并填充来自 **联系人** 直接签入协议。
