# 创建适合移动设备的视图

了解如何使用“移动焦点”模式创建文档的移动友好视图。 发件人现在可以无缝地生成一个对移动设备友好的表单，而无需任何开发人员支持。

借助移动焦点，发件人可以从以下各项中受益：

* 提高完成率：提高表单完成率，从而改善成果
* 减少时间和成本：避免将文档转换为便于移动使用的表单所需的工作
* 无需任何技术专业知识：只需花费最少精力即可生成动态、适合移动设备的表单
